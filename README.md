# 🚀 Space Mission Control

A modern full-stack Mission Control dashboard featuring live rocket launches, satellite tracking, interactive 3D Earth visualization, AI-powered mission explanations, and real-time space analytics. 

## Tech Stack:

- React + Vite
- TypeScript
- Tailwind CSS
- FastAPI
- PostgreSQL
- React Three Fiber
- Three.js
